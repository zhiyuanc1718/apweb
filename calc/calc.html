Limits Basics

Direct Substitution

$$let\ f\left(x\right) = {\frac {x^{2}-1}{x-1}}$$

$$\lim _{x\to 3}f\left(x\right) =4$$

Read this as "The Limit as X approaches 3 for the function f(x)"

$$\lim _{x\to 3}f\left(x\right) = {\frac {3^{2}-1}{3-1}} = 4$$

How do we solve that limit?

Since a limit is the y value that a function approaches when it approaches a x value, simplify plug in the x value.

When you plug in 3 for X, you get 8 on the numerator and 2 on the denominator. The Limit equals 4.

$$\lim _{x\to 1}f\left(x\right) =2$$

Note that when we simply plug in X = 1, we cannot divide by 0. We need to find another way to solve these limits.

Limit by Factorization

$$f\left(x\right) = {\frac {x^{2}-1}{x-1}}$$

$$\lim _{x\to 1}f\left(x\right) =2$$

How? Well, If you factor the numerator

$$\lim _{x\to 1}f\left(x\right) = {\frac {(x+1)(x-1)}{x-1}}$$

See the common factor of (x-1) ? We can cancel them out and then substitute 1 for X again

$$\lim _{x\to 1}f\left(x\right) = x+1$$

$$\lim _{x\to 1}f\left(x\right) = 1+1 = 2$$

Note: Always attempt to factor if possible

Limit using Trigonometry

$$f\left(x\right) = {\frac {sin^{3}(x)cos(x)}{1-cos^{2}x}}$$

$$\lim _{x\to 0}f\left(x\right) = ?$$

Let's use direct substitution first and see what happens

$$\lim _{x\to 0}f\left(x\right) = {\frac {sin^{3}(0)cos(0)}{1-cos^{2}(0)}} = {\frac {0}{0}}$$

When you have 0/0, this is called the Indeterminate form

You should not make any conclusions about an answer when you approach 0/0

$$[unit-circle]\ sin^{2}x + cos^{2}x = 1$$

$${\frac {sin^{3}(0)cos(0)}{sin^{2}(0)}} = {sin(0)cos(0)} = 0$$

Note: When dealing with trig functions, use your trig identities!

One Sided Limits

Left Hand

$$let\ f\left(x\right) = {\frac {1}{x}}$$

$$\lim _{x\to 0^{-}}f\left(x\right)$$

The - sign next to the 0 indicates that we are looking at the limit from only the left side. As we move towards the x value from the left, what is the limit?

Note: Consider graphing or picturing the graph in your head!

Right Hand

$$let\ f\left(x\right) = {\frac {1}{x}}$$

$$\lim _{x\to 0^{+}}f\left(x\right)$$

The + sign next to the 0 indicates that we are looking at the limit from only the right side. As we approach the x value from the right, what is the limit?

Practice 1

$$let\ g\left(x\right) = {\frac{x-2}{\left|x-2\right|}}$$

$$\lim _{x\to 2^{-}}g\left(x\right) = ?$$

Think about it and Checkout the Answer or Graph

Using direct substitution, the limit becomes 0/0 which is what we want to avoid. Looking at g(x), the magnitude of the numerator and denominator are equal: (x-2) and |x-2| both contain "x-2"

Without considering the absolute value, the limit would be 1.

$$let\ r\left(x\right) = {\frac{x-2}{x-2}} = 1$$

However, the absolute value brings the issue. We know as x approaches 2 from the left for x-2, the y value increases until it equals 0. Conclusion : x-2 is negative until x = 2

Remember that all absolute values are positive? so |x-2| is always positive until x = 2

Conclusion : the limit as g(x) approaches x = 2 from the left is -1

$$\lim _{x\to 2^{-}}g\left(x\right) = -1 $$

Practice 2

$$let\ g\left(x\right) =\frac{x^2-4x}{\left|x-4\right|}$$

$$\lim _{x\to 4^{+}}g\left(x\right) = ?$$

Try to answer on your own

Using direct substitution, the limit becomes 0/0. Factoring g(x), we have x(x-4) as numerator and |x-4| as denominator.

Without considering the absolute value, the limit would equal to x (in this case x = 4).

$$let\ r\left(x\right) = {\frac{x(x-4)}{(x-4)}} = x$$

We know as x approaches 4 from the right for x-4, the y value decreases until it equals 0. Conclusion : x-4 is positive until x = 4

All absolute values are positive, so |x-4| is always positive until x = 4

Conclusion : the limit as g(x) approaches x = 4 from the right is 1 times x which simply equals to whatever x is.

$$\lim _{x\to 4^{+}}g\left(x\right) = x = 4 $$

Limit at Infinity

Introduction

When the limit approaches infinity for x, essentially, you are finding the horizontal asymptote (if there is one).

Think of an exponential graph, the basic horizontal asymptote is y = 0. If we have a positive standard exponential graph, the horizontal asymptote is y = 0. So as x approaches negative infinity, we have a limit of 0.

$$for\ f\left(x\right) = e^x$$

$$\lim _{x\to -∞}f\left(x\right) = 0 $$

There are a few tips to solving these limits. Always consider if x is approaching negative infinity or positive infinity.

Rule 1 : exponents

For now, let's focus only on positive infinity. for any non-polynomial function such as :

$$let\ f\left(x\right) = {\frac {3x-2}{10x^2}}$$

What is the limit as x approaches infinity ?

Compare the function from the numerator to the denominator.

Why is the limit 0 when denominator's function is larger in exponent? Well if we substitute infinity for x

$$\lim _{x\to ∞}f\left(x\right) = {\frac {1}{∞}} $$

One in a billion is basically zero.

Infinity Rule Summary

If denominator's exponent is larger, we approach a limit of 0

If numerator's exponent is larger, the function increases without limits (limit Does Not Exist [DNE])

I hope you ask "what if they are the same?" : the limit is based on the coefficients (shown below).

$$\lim _{x\to ∞}{\frac {3x^{2}-1}{5x^{2}+x}} = {\frac {3}{5}}$$

Practice

$$\lim _{x\to ∞}{\frac {2x^{e}-1}{2x^{-2}+5x}} = ?$$

The answer is DNE or Limit Does Not Exist.

The function just keeps on increasing approaching infinity (limitless)

The numerator's main function is x to the e power. The denominator's exponent is -2 which is smaller than e (2.718.. Euler's number) Therefore, we conclude that the function approaches y = infinity as x approaches infinity.

Having a limit as infinity is the same as DNE.

Continuity

Continous Functions

When the function is defined at the point where the left hand and right hand limits meet one another, the function is then considered Continous at this point.

$$f\left(c\right) = \lim _{x\to c^{+}}f\left(x\right) = \lim _{x\to c^{-}}f\left(x\right)$$

If one of these do not equal, then the function is not continous at this point where x = c.

When the left hand and right hand limit both equal one another (an actual number) then the simpler equation comes :

$$f\left(c\right) = \lim _{x\to c}f\left(x\right)$$

Discontinuity

What are the types of discontinuities?

Removable discontinuity : The function becomes discontinuous due to the existence of a hole (undefined point). The limit approaches the hole and the hole could be removed to bring continuity.

Infinite (nonremovable) discontinuity : This type of discontinuity involves an asymptote which the function is undefined for this vertical asymptote. The limit of the function will not exist at an asymptote (left and right limits can exist).

Jump discontinuity : The function from the left approaches a different limit compared to the right hand’s limit.

Derivatives Basics

Disguised Derivatives

Note : Before learning derivatives, go learn limits.

Derivatives are slopes of tangent lines. The basic mathematical definition of a derivative is written in terms of limits:

The Difference Quotient

$$\lim _{h\to 0}{\frac {f(x+h)-f(x)}{h}}$$

In the definition, h represents the change in x [delta x] The derivative is based on an infinitesimal change (in other words, delta x or change in x value is zero)

When we calculate slope with algebra, we use two points to form what we call a secant line. We take the change in Y and divide by the change in X. In calculus, we take the instantaneous slope which is the slope at one point. This can be done by drawing tangent lines, but we will calculate with calculus.

First Derivative

You want to find the derivative (slope) of a standard parabola.

$$let\ q\left(x\right) = {x^{2}}$$

notation : d/dx means to take the derivative with respect to variable x

$${\frac {d}{dx}} q\left(x\right) = \lim _{h\to 0}{\frac {(x+h)^{2}-x^{2}}{h}}$$

$${\frac {(x+h)^{2}-x^{2}}{h}} = {\frac {x^{2}+2xh+h^{2}-x^{2}}{h}}$$

$${\frac {d}{dx}} q\left(x\right) = \lim _{h\to 0}2x + h = 2x$$

How is this useful? Well, if we are given any x value, we will know the slope at that point on the graph.

Power Rule

Using the difference quotient, find t'(x)

$$let\ t\left(x\right) = {x^{10}}$$

You may go off foiling with such high exponent, but there's the power rule to do this derivative.

The Power Rule can be established through constantly applying the difference quotient on functions like quadratics and cubics. The result shows that for a function with a constant as its exponent , the derivative is :

$$f'\left(x\right) = n{x^{n-1}}$$

For our polynomial t(x), the derivative using the power rule is :

$$t'\left(x\right) = 10{x^{9}}$$

Chain Rule

The chain rule applies when multiple functions are contained within a single function such as cos(2x). The inside function is 2x and the outside function is cos().

You have to take the derivative of the inside and multiply it to the derivative of the outside with the inside unchanged.

$${\frac {d}{dx}}f\left(g\left(x\right)\right) = g'\left(x\right)f'\left(g\left(x\right)\right)$$

$$let\ c\left(x\right) = cos({4x^{8}})$$

For composite function c(x), the derivative using the chain rule (and power rule) is :

$$c'\left(x\right) = -sin({4x^{8}})32x^{7}$$

Product Rule

The product rule applies when two functions are multiplied by one another. So in the case of 2xcos(x), 2x is its own function and cos(x) is its own function.

$$let\ p\left(x\right) = {x^{3}cos(x)} $$

Product Rule states that the derivative for multiplying functions is :

$${\frac {d}{dx}}f\left(x\right)g\left(x\right) = f'\left(x\right)g\left(x\right) + f\left(x\right)g'\left(x\right)$$

$$p'\left(x\right) = 3x^{2}cos(x) + (x^{3})(-sin(x))$$

Quotient Rule

The quotient rule is similar to the product rule but mainly used when two functions divide such as (2x)/Cos(x).

$$let\ Q\left(x\right) = {\frac {sin(x)}{cos(x)} = tan(x)} $$

Quotient Rule states that the derivative for dividing functions is :

$${\frac {d}{dx}}{\frac {f\left(x\right)}{g\left(x\right)}} = {\frac {f'\left(x\right)g\left(x\right) - f\left(x\right)g'\left(x\right)}{(g\left(x\right))^{2}}}$$

$$Q'\left(x\right) = {\frac {cos^{2}(x) + sin^{2}(x)}{cos^{2}(x)}} = sec^{2}(x)$$

Interpreting Derivatives

F(x) and F'(x)

The relationship between the function and slope is that if the F(x) is increasing, then F'(x) must be positive and vice versa is true.

By knowing that F(x) is positive, you cannot make any assumptions about its slope F'(x).

F'(x) and F''(x)

F''(x) means the Second Derivative or the derivative of the first derivative.

This might sound weird, but it is the slope of the slope.

F'(x) and F''(x) shares the same relationship just like that between F(x) and F'(x)

F(x) and F''(x)

The original function and its second derivative are related by concavity.

When F''(x) is positive, we have a concave up (convex) F(x).

When F''(x) is negative, we have a concave down function

Points of Inflection (POI)

When F''(x) equals zero or undefined and that F''(x) changes signs, this indicates a Point of Inflection

Points of inflection help determine a change in concavity which helps us visualize the shape of a graph.

Critical Values

When F'(x) equals zero or undefined

All critical values are potential relative minimums or maximums. If not either, then that would considered a tangency line

Derivative Theorems

Intermediate Vaule Theorem

This theorem essentially states that a continous function will have a intermediate value between two Y values.

In simpler terms, if F(1) = 5 and F(3) = 10, then between x = 1 and x = 3

there must exist a x value for which f(x) = 7 because 7 is in between of 5 and 10, and we know to get to 10 from 5, you need to go pass 7.

$$f\left(a\right) = 2 \\ f\left(b\right) = 5 \\ f\left(c\right) = 5 ≥ y ≥ 2$$

Mean Value Theorem

This theorem requires a function to be differentiable. MVT guarantees that a function will have a point between two points such that the point's derivative (slope) is equal to the average rate of change of the two points.

$${\frac {f\left(a\right) - f\left(b\right)}{a-b}} = f'\left(c\right)$$

In simple terms, this defines that there exist a point between two points where its instantaneous slope is equal to the average rate of change for a differentiable function.

Integration

Intro to Integrals

The goal of using integrals is to find the area under a curve.

Integration is the opposite of differentiation. With derivatives, you find the slope. Integrals take the area between curves and lines and are capable of producing the original function that was differentiated.

There are several basic ways to solve integrals, so lets explore!

Anti-Derivatives

Antiderivative (Indefinite Integrals) : Finding the original function given the derivative. These involve some memorization of the original function and its derivative such as seeing 2x as the derivative should remind you that the original function or the antiderivative is the quadratic x squared.

$${\frac {d}{dx}}{x^{2}} = 2x \ \ vs \int 2x\,dx = x^{2}$$

$${\frac {d}{dx}}{sin(x)} = cos(x) \ \ vs \int cos(x)\,dx = sin(x)$$

Plus C

When you take the indefinite integral of a function, you should add + C [for constant] to your answer

When taking the derivative of a function, all constants are differentiated and they become 0 (power rule).

Therefore, doing the opposite means that you will produce a constant value for which you don't know what it is (as of now).

$$\int cos(x)\,dx = sin(x)$$

The above is wrong. We should correct it to be :

$$\int cos(x)\,dx = sin(x) + C$$

Definite Integrals

Remember that integrals find the area under a curve, well now we will do some math.

$$find \ \int _{1}^{4}2x\,dx$$

The 1 and 4 represents the limits (not that limit) or the bounds of the integral. Basically, we are saying that we want the area under the curve from x = 1 to x = 4 for this function 2x

Since this is a linear function, you can easily calculate the area with trapezoids, but lets use calculus.

$${\displaystyle \int _{1}^{4}\,2x \ dx=\left[x^{2}\right]_{1}^{4}=(4)^{2}-(1)^{2}\,.}$$

$${\displaystyle \int _{1}^{4}\,2x \ dx= 16 - 1 = 15}$$

Anti-Power Rule

Remember how power rule works with derivatives?

Now we are doing the opposite.

The opposite of subtracting is adding, and the opposite of multiplying is dividing.

$$\int x^{n}\,dx = {\frac {x^{n+1}}{n+1}} + C$$

This will undo a derivative that was derived with power rule. [simple?]

Mastering this method will allow you to tackle an Integral's chain rule.

Chain Rule (U Substitution)

When we take a derivative for a composite function, we have the derivative of the inside function multiplied by the outside's derivative with the inside unchanged.

Doing the chain rule for integration is a bit tricky and the substitution rule comes into play.

$$find \ \int (2x)^{2}\,dx$$

Looking at the derivative that we are integrating, we should expect the format of "inside outside". Clearly, there is an inside function (2x) but its derivative is not here.

Follow the format of U-substitution below

$$let \ u = 2x \ so \ du = 2$$

$${\frac {1}{2}du} = dx$$

$$now \ \int (u)^{2}\,{\frac {1}{2}du}$$

$$\int (u)^{2}\,{\frac {1}{2}du} = {\frac {1}{2}}{\frac {u^{3}}{3}} + C$$

$$ = {\frac {1}{6}}*(2x)^{3} + C$$

FTC

Definition

The Fundamental Theorem of Calculus is a theorem that links the concept of differentiating a function with the concept of integrating a function. The theorem is separated into two parts.

If f is continuous on [a, b], then the function

$$F(x)=\int _{a}^{x}\!f(t)\,dt.$$

will have a derivative on the interval such that

$$F'(x)= f(x)$$

Secondly, if the function has a closed interval with defined real values, then :

$$\int _{a}^{b}f(x)\,dx=F(b)-F(a)$$

Mean Value Theorem

The MVT for integrals finds the average value of the function over an interval

$$ {\frac {1}{b-a}} \int _{a}^{b}f(x)\,dx={\frac {F(b)-F(a)}{b-a}}.$$

where F(x) is the integral of f(x). This essentially finds the average value of f(x) similar to how the slope formula finds the average rate of change.

Let's try a simple and obvious example

$$let \ f\left(x\right) = 5$$

What is the average value of f(x), well it can only be 5.

Now use calculus and prove that !

$$\int _{a}^{b}5\,dx=\left[5x\right]_{a}^{b}$$

$$ = {\frac {5(b) - 5(a)}{b-a}} = 5 $$

We just proved that the average value of a straight line y = 5 is 5.
